Prueba Practica API libros

Obtener Autores

o POST: { url }/api/autor: Crear un autor (Nombre, Nacionalidad).

o GET: {url}/api/autor: Retorna todos los autores con sus respectivos libros.

o GET: {url}/api/autor/< id >: Retorna un autor específico con sus respectivos libros.

o POST: {url}/api/libro: Crea un libro y lo asocia a un autor. Si el autor existe, deberá asociar el libro, de no ser así, el sistema deberá crear al autor automáticamente además de crear el libro. Asuma que los nombres de autores serán genéricos, por ejemplo no se encontrará en el caso donde el autor de un libro se llame J.R.R Tolkien y el de otro libro se llame Tolkien.

Retornar Autores

        GET url/api/autor/
        Content type:application/json
        URL: http://localhost:8000/

Retorna Autores por ID

        GET url/api/autor/1
        Content type:application/json
        URL: http://localhost:8000/

Crear Autor

        POST url/api/autor/
        Content type:application/json
        URL: http://localhost:8000/

Cuerpo de Solicitud:

            {
            "name": "author1",
            "nationality": "Chilena"
            }

Crear Libros

        POST url/api/libro/
        Content type:application/json
        URL: http://localhost:8000/

Cuerpo de Solicitud:

            {
            "author_id": 4,
            "name": "book8",
            "pages_number": 450
            }
